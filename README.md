# macOS Nexus (Version 10.16) 🚀

macOS Nexus is an independent open-source prototype designed to explore optimized upgrade paths for legacy Mac hardware stuck on macOS 10.15 Catalina. 

## 🛠️ Project Goals
- **Seamless Interface:** A modern, sleek system preference layout.
- **Native Utility:** Built-in secure folder structures and documentation access.
- **Efficiency:** A lightweight system environment designed to keep older intel machines running smoothly.

## Folder Structure
- `/Interface`: UI code for custom settings and preferences panels.
- `/SystemFiles`: Custom wallpapers and system PDFs.
- `/Installer`: The automated installation script.

*Note: This is an independent educational prototype built for hardware longevity research.*
